## **GitHub Showcase: DevOps & Git Tasks**

### **1. PR Workflow Automation**

**Task:** Create a Pull Request entirely from the command line.
**Solution:** Used branching, committed updates, and created a PR via GitHub CLI.
**Outcome:** Demonstrated end-to-end PR workflow without using the web interface.

---

### **2. Restore Stashed Changes**

**Task:** Recover previous in-progress work from a stash.
**Solution:** Identified the correct stash and applied it safely to the working directory.
**Outcome:** Work was restored and ready for further development without losing any progress.

---

### **3. Rebase Feature Branch**

**Task:** Update a feature branch with the latest changes from `main`.
**Solution:** Rebased the branch to maintain a clean, linear commit history.
**Outcome:** Feature branch synchronized with main, conflicts resolved efficiently.

---

### **4. Resolve Merge Conflicts**

**Task:** Local branch and remote branch diverged, causing conflicts.
**Solution:** Pulled remote changes, manually resolved conflicts, and committed the resolution.
**Outcome:** Repository successfully merged, fully synchronized, and ready for deployment.

---

### **5. PR Showcase (Final Workflow)**

**Task:** Demonstrate a complete GitHub PR workflow from start to finish.
**Solution:** Created a feature branch, implemented changes, pushed to remote, and merged via CLI.
**Outcome:** Full PR lifecycle executed cleanly, showing professional Git workflow.
